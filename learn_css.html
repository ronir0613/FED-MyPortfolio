CSS (Cascading Style Sheets)

1. CSS Basics

Inline, Internal, and External CSS

Inline CSS


<p style="color: red;">This is a red paragraph.</p>

style attribute: Adds CSS directly within an HTML element.

Internal CSS


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Internal CSS</title>
    <style>
        p {
            color: blue;
        }
    </style>
</head>
<body>
    <p>This is a blue paragraph.</p>
</body>
</html>

<style> tag: Contains CSS within the <head> section.

External CSS


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>External CSS</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <p>This is a styled paragraph.</p>
</body>
</html>
            

/* styles.css */
p {
    color: green;
}

<link> tag: Links to an external CSS file.

2. Selectors

Element, ID, Class, and Attribute Selectors

Element Selector


p {
    color: black;
}

Element Selector: Targets all <p> elements.

ID Selector


#unique {
    color: orange;
}
<p id="unique">This is an orange paragraph.</p>

ID Selector: Targets an element with a specific ID.

Class Selector


.special {
    color: purple;
}
<p class="special">This is a purple paragraph.</p>

Class Selector: Targets elements with a specific class.

Attribute Selector


a[target="_blank"] {
    color: red;
}
<a href="https://example.com" target="_blank">External Link</a>

Attribute Selector: Targets elements with a specific attribute and value.

3. Colors and Backgrounds

Color Properties

Color Names and Hex Codes


p {
    color: red; /* Color name */
    color: #ff0000; /* Hex code */
}

color: Sets the text color.

RGB and RGBA


p {
    color: rgb(255, 0, 0); /* RGB */
    color: rgba(255, 0, 0, 0.5); /* RGBA with transparency */
}

Background Properties

Background Color and Image


body {
    background-color: lightblue;
    background-image: url('background.jpg');
}

background-color: Sets the background color.

background-image: Sets the background image.

Background Position, Size, and Repeat


body {
    background-position: center;
    background-size: cover;
    background-repeat: no-repeat;
}

background-position: Sets the starting position of the background image.

background-size: Resizes the background image.

background-repeat: Controls the repetition of the background image.

4. Box Model

Margins, Padding, Borders, and Content

Box Model Structure


div {
    width: 300px;
    padding: 20px;
    border: 5px solid black;
    margin: 10px;
}

width: Width of the content area.

padding: Space between content and border.

border: Border around the element.

margin: Space outside the border.

Box Sizing


div {
    box-sizing: border-box;
}

box-sizing: Controls how the total width and height of an element is calculated.

5. Text Styling

Font Properties

Font Family, Size, and Style


p {
    font-family: 'Arial', sans-serif;
    font-size: 16px;
    font-style: italic;
}

font-family: Sets the font family.

font-size: Sets the font size.

font-style: Sets the font style (e.g., normal, italic).

Font Weight and Line Height


p {
    font-weight: bold;
    line-height: 1.5;
}

font-weight: Sets the weight (boldness) of the font.

line-height: Sets the space between lines of text.

Text Alignment, Decoration, and Transform

Text Alignment and Decoration


p {
    text-align: center;
    text-decoration: underline;
}

text-align: Sets the horizontal alignment of text.

text-decoration: Adds decoration to text (e.g., underline, overline, line-through).

Text Transform


p {
    text-transform: uppercase;
}

text-transform: Controls the capitalization of text.

6. Layouts

Display Property

Block, Inline, and Inline-Block


div {
    display: block;
}
span {
    display: inline;
}
img {
    display: inline-block;
}

display: Controls the display behavior of an element.

Positioning

Static, Relative, Absolute, Fixed, and Sticky


div {
    position: static; /* Default */
}
.relative {
    position: relative;
    top: 10px;
    left: 10px;
}
.absolute {
    position: absolute;
    top: 20px;
    left: 20px;
}
.fixed {
    position: fixed;
    top: 0;
    left: 0;
}
.sticky {
    position: sticky;
    top: 0;
}

position: Specifies the type of positioning method used for an element.

Flexbox

Flex Container and Items


.container {
    display: flex;
    justify-content: space-between;
    align-items: center;
}
.item {
    flex: 1;
    padding: 10px;
    background-color: lightgrey;
}

display: flex: Defines a flex container.

justify-content: Aligns items horizontally.

align-items: Aligns items vertically.

flex: Defines the flex grow, shrink, and basis properties of a flex item.

Grid Layout

Grid Container and Items


.container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    grid-gap: 10px;
}
.item {
    background-color: lightgrey;
    padding: 20px;
}

display: grid: Defines a grid container.

grid-template-columns: Defines the columns of the grid.

grid-gap: Sets the gap between grid items.

7. Responsive Design

Media Queries

Basic Media Query


@media (max-width: 600px) {
    body {
        background-color: lightcoral;
    }
}

@media: Applies styles based on specific conditions (e.g., screen size).

Responsive Font Sizes


body {
    font-size: 16px;
}
@media (max-width: 600px) {
    body {
        font-size: 14px;
}

Adjusts font size based on screen size.

8. CSS Animations and Transitions

Transitions

Basic Transition


div {
    transition: background-color 0.5s ease;
}
div:hover {
    background-color: lightblue;
}

transition: Adds transition effects to elements.

Animations

Keyframes and Animation


@keyframes slidein {
    from {
        transform: translateX(-100%);
    }
    to {
        transform: translateX(0);
    }
}
div {
    animation: slidein 2s forwards;
}

@keyframes: Defines the animation.

animation: Applies the animation to an element.

9. Pseudo-classes and Pseudo-elements

Pseudo-classes

Hover, Focus, and Active


a:hover {
    color: red;
}
input:focus {
    border-color: blue;
}
button:active {
    background-color: green;
}

:hover: Applies styles when the element is hovered.

:focus: Applies styles when the element is focused.

:active: Applies styles when the element is active.

Pseudo-elements

Before and After


p::before {
    content: "Note: ";
    font-weight: bold;
}
p::after {
    content: " - End of note.";
    font-weight: bold;
}

::before: Inserts content before an element.

::after: Inserts content after an element.

10. CSS Variables

Defining and Using Variables


:root {
    --primary-color: #3498db;
    --secondary-color: #2ecc71;
}
body {
    color: var(--primary-color);
}
h1 {
    color: var(--secondary-color);
}

--variable-name: Defines a custom property (variable).

var(--variable-name): References the variable.
